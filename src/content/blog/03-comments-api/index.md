title: "اتصال دیدگاه‌ها به API خارجی"
description: "خواندن و ثبت دیدگاه با استفاده از یک سرویس REST عمومی."
date: "2026-02-24"
---

در صفحه هر نوشته، بخش دیدگاه‌ها به شکل مستقیم به سرویس خارجی متصل می‌شود.

جریان کار به این صورت است:

- دریافت دیدگاه‌ها با `GET /comments?postSlug=...`
- ثبت دیدگاه جدید با `POST /comments`
- نمایش وضعیت بارگذاری، خطا و موفقیت به کاربر

این ساختار باعث می‌شود بدون نیاز به پایگاه‌داده داخلی در Astro، بخش دیدگاه‌ها به راحتی مدیریت شود.
