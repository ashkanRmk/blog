## تیتر سطح دو

این یک پاراگراف ساده است که از **متن بولد**، *متن ایتالیک* و ~~متن خط‌خورده~~ پشتیبانی می‌کند.

برای اشاره کوتاه به دستور از `pnpm run dev` استفاده می‌کنیم.
